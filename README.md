# PathForge AI

PathForge AI is an AI-powered career intelligence system that analyzes skill gaps and generates personalized learning roadmaps.

## Problem
Students often feel confused about what skills to learn for their dream careers due to unstructured learning resources.

## Solution
PathForge AI uses a dynamic skill graph and AI analysis to:
- Detect missing skills
- Rank them by priority
- Generate a structured learning roadmap

## Tech Stack
- AWS Amplify
- API Gateway
- AWS Lambda
- AWS Bedrock
- DynamoDB

## Core Features
- Skill gap analysis
- Dynamic skill graph modeling
- Personalized learning plans
- Multilingual support
- Low-bandwidth design
